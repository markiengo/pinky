# 08 — Roadmap

---

## Phase 1 — Competition MVP

- Frontend-first React SPA.
- Mock anonymous session.
- Intake survey with 3 core categories.
- Theme packs.
- 1:1 matched chat.
- Free one-match limit.
- Upgrade prompt.
- Light safety/report.

---

## Phase 2 — Real MVP

- Real auth/session.
- Real backend for profiles, surveys, matches, conversations.
- Real-time chat via WebSocket/Supabase/Firebase.
- Block/report persistence.
- Basic admin review for reports.
- Payment integration.

---

## Phase 3 — Product Expansion

- More categories.
- More psychological self-reflection tests.
- Emotion tracker/journal.
- Confession wall.
- Support circles.
- Direct counselor consultation.
- School/organization partnerships.

---

## Deferred Technical Decisions

- Backend stack.
- Database provider.
- Real-time chat provider.
- Payment provider.
- Moderation architecture.
- Counselor scheduling workflow.
